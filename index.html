    Hotel Booking API

    Major Technologies Used: Node.js, Express.js, PostgreSQL, Prisma ORM, JWT, Swagger, Postman

    The Hotel Booking API is a backend-focused, RESTful application designed to handle hotel room management and booking operations in a secure and scalable manner. It supports core features such as room and booking CRUD operations, user authentication, and role-based access control. Built using a clean, layered architecture with Node.js and Express, the API leverages PostgreSQL and Prisma ORM for efficient data modeling and database management. Security is implemented using JWT-based authentication, and all endpoints are tested and documented using Postman and Swagger, ensuring reliability and ease of integration.
